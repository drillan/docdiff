Database Schema

Nodes Table

Stores document structural elements:

Nodes Table Schema
 1CREATE TABLE nodes (
 2    id TEXT PRIMARY KEY,
 3    type TEXT NOT NULL,
 4    content TEXT NOT NULL,
 5    level INTEGER,
 6    title TEXT,
 7    label TEXT,
 8    name TEXT,
 9    caption TEXT,
10    language TEXT,
11    parent_id TEXT,
12    children_ids TEXT,
13    file_path TEXT NOT NULL,
14    line_number INTEGER NOT NULL,
15    content_hash TEXT NOT NULL,
16    metadata JSON,
17    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
18    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
19    FOREIGN KEY (parent_id) REFERENCES nodes(id) ON DELETE CASCADE
20);

Indexes:

  • label and name: Fast reference resolution

  • parent_id: Efficient hierarchy traversal

  • file_path: File-based operations

  • content_hash: Change detection

References Table

Maintains cross-reference relationships:

References Table Schema
1CREATE TABLE references (
2    id INTEGER PRIMARY KEY AUTOINCREMENT,
3    source_node_id TEXT NOT NULL,
4    target_label TEXT NOT NULL,
5    ref_type TEXT NOT NULL,
6    resolved_target_id TEXT,
7    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
8    FOREIGN KEY (source_node_id) REFERENCES nodes(id) ON DELETE CASCADE
9);

Fields:

  • source_node_id: Node containing the reference

  • target_label: Label being referenced

  • ref_type: Type of reference (e.g., ‘ref’, ‘doc’, ‘numref’)

  • resolved_target_id: Resolved target node ID (if found)

Translation Units Table

Tracks translation status and content:

Translation Units Table Schema
 1CREATE TABLE translation_units (
 2    id INTEGER PRIMARY KEY AUTOINCREMENT,
 3    node_id TEXT NOT NULL,
 4    source_lang TEXT NOT NULL,
 5    target_lang TEXT NOT NULL,
 6    source_content TEXT NOT NULL,
 7    translated_content TEXT,
 8    status TEXT DEFAULT 'pending',
 9    content_hash TEXT NOT NULL,
10    translation_date TIMESTAMP,
11    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
12    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
13    FOREIGN KEY (node_id) REFERENCES nodes(id) ON DELETE CASCADE,
14    UNIQUE(node_id, source_lang, target_lang)
15);

Indexes:

  • status: Filter by translation state

  • (node_id, source_lang, target_lang): Unique constraint for language pairs
